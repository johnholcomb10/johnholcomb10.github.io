# My Portfolio Website Repository

John Holcomb (johnholcomb10)

## About this Website

This is my portfolio website, made as a project for my CSC 4330 (Software Systems Development) course at LSU, highlighting my job experience, education, projects, and awards.. It was created using standard HTML/CSS and Bootstrap, a CSS framework for responsive, front-end web development. A Sass pipeline was implemented to automatically compile the .scss file into .css on deployment, allowing for Bootstrap classes to be applied within CSS files. I plan on updating the site regularly, and the date of last update is present in the welcoming alert and the footer.

### Browsers Tested on:

- Chrome
- Edge
- Firefox
- Avast Secure Browser

### Resolutions Designed for:

- 360 px - 1920 px width
